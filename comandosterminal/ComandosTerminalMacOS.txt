Como compilar um programa escrito em linguagem C no terminal do macOS 

===COMANDOS TERMINAL macOS===

*Com o compilador clang:

clang --version

clang seu_codigo.c -o seu_programa

./seu_programa

*Com compilador gcc

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"

brew update

brew install gcc

gcc --version
